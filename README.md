# Agentic Video Analysis

This repository contains an agent-based pipeline for analyzing video/comment data end-to-end:

- Phase 1: Literature review
- Phase 2: Data preparation & EDA
- Phase 3: Architecture & message schemas
- Phase 4: Agent implementations (multi-agent pipeline)
- Phase 5: Evaluation (metrics, ablations, error analysis)
- Phase 6: Report & presentation assets

## Quick start

1. Create a virtual environment.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the pipeline (placeholder): `python scripts/run_pipeline.py`

## Project layout

- `src/` Python source code
- `data/` datasets and artifacts (raw/clean are placeholders and should be gitignored if you add content)
- `docs/` scientific documentation
- `notebooks/` reproducible exploration and experiments
- `tests/` unit/integration tests skeleton
- `evaluation/` scripts for scientific evaluation
- `scripts/` runnable entry points
